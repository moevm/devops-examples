# Monkey Testing — пример

Веб-приложение «Таск-менеджер» для демонстрации monkey testing с помощью [gremlins.js](https://github.com/marmelab/gremlins.js).

## Запуск

1. Скопируйте файл с переменными окружения и задайте свои значения:

```bash
cp .env.example .env
```

2. Запустите сервисы:

```bash
docker compose up --build
```

Приложение доступно на http://localhost:5000

## Как запустить monkey testing

1. Откройте http://localhost:5000 в браузере
2. В правом нижнем углу находится панель **Monkey Testing**
3. Настройте параметры:
   - **Действий в секунду** — частота случайных действий (1–30)
   - **Клики** — случайные клики по элементам
   - **Заполнение форм** — случайный ввод в поля форм
   - **Скроллинг** — случайный скроллинг страницы
4. Нажмите **▶ Старт** — gremlins начнут случайно взаимодействовать с интерфейсом
5. Нажмите **■ Стоп** для остановки

## Что тестируется

- Создание, редактирование, удаление задач
- Фильтрация по приоритету и статусу
- Смена статуса задачи
- Добавление комментариев
- Полнотекстовый поиск
- REST API (`GET /api/tasks`, `DELETE /api/tasks/<id>`)

## Описание шаблонов приложения (каталог templates)

- base.html - подключение gremlins.js + панель управления
- index.html - список задач с фильтрами
- task_form.html - форма создания/редактирования
- task_detail.html - детали задачи + комментарии
- search.html - поиск
